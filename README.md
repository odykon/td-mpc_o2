#TD-MPC with Optimized Optimization: Efficient TD-MPC for continuous control through a latent action space learned with model-based RL
My thesis project at ECE-NTUA.
We extend TD-MPC with a learned latent action space making planning x16 times more efficient. The latent space is state-dependent and learned end-to-end with Reinforcement Learning through the Differentiable CEM. We also provide an implemenation of DCEM for continuous control.
